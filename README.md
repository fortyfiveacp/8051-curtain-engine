# Curtain Engine

An ECS C++ game engine specializing in 2D bullet hell games.

## 「東方巫女戦場 ～ Double Prayer Duel」

A Touhou fan game made using Curtain Engine, featuring the series' signature danmaku 2D shoot'em up gameplay.

Reimu has won the lottery and amassed a great fortune. Sanae learns of this and decides to take shrine rivalry
seriously. Reimu is left to defend the shrine from an onslaught of Moriya Shrine-affiliated fairies before
confronting Sanae herself.

### Features
- One stage & boss battle
- Weave through intricate bullet patterns
- Manage your lives, bombs, and power
- Enjoy a rearrangement of the official Touhou soundtrack

## Technologies & Tools

- C++ & CMake
- SDL
- Texturepacker
- Aseprite, Photoshop, Clip Studio Paint
- FL Studio
